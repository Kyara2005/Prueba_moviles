# Proyecto Prueba 

## 👤 Nombre del estudiante:
Kyara Altamirano

## Nombre del proyecto:
photo-gallery

## Tema seleccionado:
Consumo de API libros con Ionic + Angular

## API utilizada:
Open Library API  
https://openlibrary.org/developers/api

### Ejemplo de endpoint:
https://openlibrary.org/search.json?q=harry+potter

## 🛠️ Tecnologías:
- Ionic Framework
- Angular
- TypeScript
- HTML / SCSS
- Node.js / npm

## Instrucciones de instalación:

```bash
git clone https://github.com/Kyara2005/Prueba_moviles.git
cd photo-gallery
npm install
```
